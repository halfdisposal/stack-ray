# Stack-RAY
A tile stacking game in raylib and rust

# Dependencies
- Raylib
- Rand
- Chrono
- and ofcourse Rust

# Build From Source
```bash
cargo add raylib
cargo add rand
cargo add chrono

git clone https://github.com/halfdisposal/stack-ray.git
cd stack-ray
cargo build --release
cargo run
```
[!Note] So far windows release in mingw toolchain.

[!Note] MSVC toolchain and Linux/Mac should build from source.
